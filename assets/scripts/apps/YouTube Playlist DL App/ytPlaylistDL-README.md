# youtubePlaylistDL

A simple script to download all videos in a youtube playlist (720p quality)

## Requirements

```bash
clear;
python3 -m pip install --upgrade pip;
python3 -m pip install --upgrade setuptools;
pip3 install pytube;
```

## Usage

```bash
python3 ytPlaylistDL.py <playlistURL> <destinationPath>;
```
